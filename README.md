# image-converter

util to convert images to webp format ready to use in any project

## how to use
- install imagemagick
- - ```brew install imagemagick```
- Make the file an executable
-  ```chmod +x image-converter.sh```
- located in the same folder as the images you want to convert run: ```./convert-to-webp.sh```
- it will generate a folder called webp with images in different standard sizes ready for production.
- you can also pass the directory where the iamges are.
